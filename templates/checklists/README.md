# Checklists

Quick-reference lists for reviewing key topics before or during ML system design interviews.


## Available Checklists

- Interview Q&A Checklist -- common ML system design questions organized by category
- Failure Modes Checklist -- failure scenarios to discuss proactively in interviews
- Cost and Latency Checklist -- key cost drivers and latency considerations
- Eval Metrics Checklist -- evaluation metrics to mention for different ML task types


## How to Use During Revision

- Work through each checklist and mark items you can confidently explain vs. items that need more study
- Before an interview, scan the relevant checklists to refresh your mental index of topics to cover
