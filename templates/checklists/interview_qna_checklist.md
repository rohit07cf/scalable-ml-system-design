# Interview Q&A Checklist

Common ML system design questions grouped by category. Use the checkboxes to track which ones you can answer confidently.


## Ranking and Recommendation

- [ ] Design a news feed ranking system
- [ ] Design a product recommendation engine for e-commerce
- [ ] Design a search ranking system with personalization
- [ ] Design an ad click prediction system


## NLP and LLM Systems

- [ ] Design a question-answering system over a large document corpus
- [ ] Design a real-time content moderation pipeline
- [ ] Design a chatbot with retrieval-augmented generation
- [ ] Design an LLM-powered summarization service


## Computer Vision

- [ ] Design an image search system
- [ ] Design a content-based image moderation system
- [ ] Design a real-time object detection pipeline for video
- [ ] Design an OCR system for document processing


## Data and Infrastructure

- [ ] Design a feature store for a large ML platform
- [ ] Design an ML model serving infrastructure
- [ ] Design an experimentation platform for A/B testing ML models
- [ ] Design a data labeling and annotation pipeline
